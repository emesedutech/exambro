Cara Kerja

  1. Isi token GitHub & nama repo
  2. Upload logo & atur tampilan splash
  3. Tentukan warna & nama aplikasi
  4. Set PIN pengawas yang kuat
  5. Klik Build APK Sekarang
  6. Tunggu ~3 menit, unduh APK
Pastikan .github/workflows/build.yml dan scripts/apply_config.py sudah ada di repo Anda.
